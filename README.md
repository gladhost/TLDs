# TLDs

TLDs is a multi-language library to retrieve and validate domain's TLDs.

## Supported languages

- [Node & Typescript](./node/README.md)
- [Python](./python/README.md)

## Functions

- `getTlds() -> string[]`
- `isValidTld(tld:string) -> boolean`
- `hasDomainValidTld(domain:string) -> boolean`
